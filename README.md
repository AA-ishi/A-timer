# Cute Timer（A-timer）

まあるい形の、音がしないデスクトップ用タイマーアプリです。
時間になるとキラキラ光るので画面に集中していても
時間が来たことに気が付けます。
発表の時にキラキラが見えたら時間が来たことがわかります。
作業・勉強・休憩など、音が鳴らないからこその使い方はあなた次第。

---

## ダウンロード方法（PC初心者でもOK）自己責任でお願いします

Cute Timer には **インストール版** と **ポータブル版** の2種類があります。
使い方に合わせて、どちらかを選んでください。

---

## 📌 インストール版（おすすめ）
**Cute Timer Setup 1.0.0.exe**

・インストールして使う通常版です  
・スタートメニューに登録されます  
・タスクバーにピン留めできます  
・PC初心者の方はこちらがおすすめです

### インストール方法
1. このページ右側の **Releases** から「最新バージョン」を開きます  
2. **Cute Timer Setup 1.0.0.exe** をクリックしてダウンロードします  
3. ダウンロードしたファイルをダブルクリックしてインストールします  
4. インストール後、スタートメニューから起動できます

---

## 📁 ポータブル版（インストール不要）
**CuteTimer-PT.exe**

・ダウンロードしてそのまま使える軽量版です  
・タスクバーにピン留めできない場合があります（Windows の仕様）  
・アプリ本体を好きな場所に置いて使えます

### 使い方
1. Releases の「最新バージョン」を開きます  
2. **CuteTimer-PT.exe** をクリックしてダウンロードします  
3. ダウンロードしたファイルをダブルクリックすると起動します  
   （インストール不要・すぐ使えます）

---

## 📝 注意
Windows が「不明なアプリです」と表示する場合がありますが、  
右下の **「詳細情報」→「実行」** を押すと起動できます。

---


## 🕒 **　アプリの使い方　**

### ① アプリを起動する
ダウンロードした `cute-timer 1.0.0.exe` をダブルクリックするだけ。

### ② 時間をセットする(カウントダウン）
画面の数字をクリックして、  
- 分  
- 秒  
を入力します。

### ③ スタート
「開始」ボタンを押すとカウントダウンが始まります。

<img width="450" height="409" alt="image" src="https://github.com/user-attachments/assets/c874b295-c2df-491d-a51c-9031f7bf99e5" />

### ④ 時間をセットする(カウントアップ）
「リセット」ボタンをダブルクリック、  
- 分 00 
- 秒 00
と表示されます。

### ⑤ スタート
「開始」ボタンを押すとカウントアップが始まります。

### ⑥ 一時停止
「一時停止」ボタンを押すとカウントアップが一時停止します。

### ⑦ リセット
「リセット」ボタンを押すとカウントアップがリセットされます。
