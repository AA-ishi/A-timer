# Cute Timer（A-timer）

透明で、ほっこり使えるデスクトップ用タイマーアプリです。  
作業・勉強・休憩など、ちょっとした時間管理に使えます。

---

## 📥 ダウンロード方法（PC初心者でもOK）

1. このページの右側にある **Releases** の「最新バージョン」を開きます  
2. 一覧の中から  
   **`cute-timer 1.0.0.exe`**  
   をクリックしてダウンロードします  
3. ダウンロードしたファイルを **ダブルクリック** するとアプリが起動します  
   （インストール不要・すぐ使えます）

※ Windows が「不明なアプリです」と表示する場合がありますが、  
　右下の **「詳細情報」→「実行」** を押すと起動できます。

---

## 🕒 使い方

### ① アプリを起動する
ダウンロードした `cute-timer 1.0.0.exe` をダブルクリックするだけ。

### ② 時間をセットする
画面の数字をクリックして、  
- 分  
- 秒  
を入力します。

### ③ スタート
「Start」ボタンを押すとカウントダウンが始まります。

### ④ 一時停止・リセット
- **Pause**：一時停止  
- **Reset**：時間を元に戻す

### ⑤ 設定画面
右上の「⚙」ボタンから設定できます。

設定できる内容：
- アプリの透明度  
- 色のテーマ  
- 表示サイズ  
- 音のオン/オフ
