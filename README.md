# Cute Timer（A-timer）

透明感のある、音がしないデスクトップ用タイマーアプリです。
時間になるとキラキラ光るので画面に集中していても
時間が来たことに気が付けます。
発表の時にキラキラが見えたら時間が来たことがわかります。
作業・勉強・休憩など、音が鳴らないからこその
使い方はあなた次第で時間管理に使えます。

---

## 📥 ダウンロード方法（PC初心者でもOK）

1. このページの右側にある **Releases** の「最新バージョン」を開きます  
2. 一覧の中から  
   **`cute-timer 1.0.0.exe`**  
   をクリックしてダウンロードします  
3. ダウンロードしたファイルを **ダブルクリック** するとアプリが起動します  
   （インストール不要・すぐ使えます）

※ Windows が「不明なアプリです」と表示する場合がありますが、  
　右下の **「詳細情報」→「実行」** を押すと起動できます。

---

## 🕒 使い方

### ① アプリを起動する
ダウンロードした `cute-timer 1.0.0.exe` をダブルクリックするだけ。

### ② 時間をセットする
画面の数字をクリックして、  
- 分  
- 秒  
を入力します。

### ③ スタート
「Start」ボタンを押すとカウントダウンが始まります。
